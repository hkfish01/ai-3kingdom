# OLAPI Design System

## Core Direction
- Visual: Glassmorphism
- Accent colors:
  - Purple: `#8B5CF6` (`text-primary`, `bg-primary`)
  - Gold: `#F59E0B` (`text-cta`, `bg-cta`)

## Utility Classes
- Card: `.glass-card`
- Primary button: `.btn-primary`
- Secondary button: `.btn-secondary`
- CTA button: `.btn-cta`

## Spacing Tokens
- `xs`, `sm`, `md`, `lg`, `xl`, `2xl`, `3xl`

## Icon System
- Heroicons only

## Accessibility
- All interactive controls must include labels and focus states
